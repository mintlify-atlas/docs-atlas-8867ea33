<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-semibold dark:text-white text-gray-900 mb-4">Quick start</h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">Get your development environment up and running in minutes</p>
  
  <Steps>
    <Step title="Clone the repository">
      Clone the Jasper Pet Shop repository to your local machine:
      
      ```bash
      git clone https://github.com/SaulSneider7/jasperpetshop.git
      cd jasperpetshop
      ```
    </Step>
    
    <Step title="Install dependencies">
      Install all required packages using npm:
      
      ```bash
      npm install
      ```
      
      This will install React, Vite, TailwindCSS, React Router, and all other dependencies.
    </Step>
    
    <Step title="Start the development server">
      Launch the Vite development server:
      
      ```bash
      npm run dev
      ```
      
      Your app will be available at `http://localhost:5173`
    </Step>
    
    <Step title="Build for production">
      When you're ready to deploy, create an optimized production build:
      
      ```bash
      npm run build
      ```
      
      The compiled assets will be in the `dist/` directory.
    </Step>
  </Steps>
</div>
