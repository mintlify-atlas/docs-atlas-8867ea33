## Overview

The `ProductCard` component displays individual product information with size variant selection, pricing, and cart integration. It features hover effects and responsive design optimized for both mobile and desktop experiences.

## Import

```tsx
import { ProductCard } from '../components/ProductCard';
import type { Product } from '../data/products';
```

## Props

<ParamField path="product" type="Product" required>
  The product object containing all product information including:
  - `id` - Unique identifier
  - `name` - Product name
  - `category` - Product category
  - `price` - Base price (string format)
  - `image` - Product image URL
  - `variants` - Optional array of size/price variants
</ParamField>

<ParamField path="delay" type="number" default="0">
  Animation delay in milliseconds for AOS (Animate On Scroll) effects
</ParamField>

## Usage

### Basic Usage

```tsx
import { ProductCard } from './components/ProductCard';
import { products } from './data/products';

function ProductGrid() {
  return (
    <div className="grid md:grid-cols-3 gap-8">
      {products.map((product) => (
        <ProductCard key={product.id} product={product} />
      ))}
    </div>
  );
}
```

### With Animation Delay

```tsx
<div className="grid md:grid-cols-4 gap-10">
  {products.map((product, i) => (
    <ProductCard 
      key={product.id} 
      product={product} 
      delay={i * 100} // Stagger animations
    />
  ))}
</div>
```

## State Management

The component manages two pieces of local state:

<ParamField path="selectedSize" type="string">
  Currently selected size variant. Defaults to the first variant's size.
</ParamField>

<ParamField path="currentPrice" type="string">
  Price of the currently selected variant. Updates when size changes.
</ParamField>

```tsx
const [selectedSize, setSelectedSize] = React.useState(
  product.variants?.[0]?.size || ''
);
const [currentPrice, setCurrentPrice] = React.useState(
  product.variants?.[0]?.price || product.price
);
```

## Component Structure

### Image Container

The product image features multiple interactive elements:

```tsx
<div className="relative aspect-[4/5] rounded-[40px] overflow-hidden
```

<Info>

### Product Info Section

<Steps>
  <Step title="Header">
    Displays category, product name, and current price
  </Step>
  <Step title="Variant Selector">
    Size buttons that update selected size and price when clicked
  </Step>
</Steps>

```tsx
<div className="flex flex-col px-2">
  {/* Header */}
  <div className="flex justify-between items-start mb-3">
    <div>
      <p className="text-[10px] uppercase text-[#D4AF37]">
        {product.category}
      </p>
      <h3 className="text-lg font-serif font-bold">
        {product.name}
      </h3>
    </div>
    <p className="text-lg font-bold">{currentPrice}</p>
  </div>

  {/* Variant Buttons */}
  {product.variants && (
    <div className="flex flex-wrap gap-2">
      {product.variants.map((v) => (
        <button
          key={v.size}
          onClick={() => handleSizeChange(v.size, v.price)}
          className={selectedSize === v.size 
            ? 'bg-[#D4AF37] text-white' 
            : 'bg-white text-[#1a1a1a]'}
        >
          {v.size}
        </button>
      ))}
    </div>
  )}
</div>
```

## Adding to Cart

The component uses the `useCart` hook to add items:

```tsx
const { addToCart } = useCart();

// When button clicked
addToCart(product, selectedSize, currentPrice);
```

<Warning>
  Always pass the selected size and current price to ensure the cart reflects the user's choices.
</Warning>

## Variant Selection

When a size variant is clicked:

```tsx
const handleSizeChange = (size: string, price: string) => {
  setSelectedSize(size);
  setCurrentPrice(price);
};
```

## Animations

The card uses AOS for entrance animations:

```tsx
data-aos="fade-up" 
data-aos-delay={delay}
```

This creates a staggered fade-up effect when multiple cards are displayed in a grid.
