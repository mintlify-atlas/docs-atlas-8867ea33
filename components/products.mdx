## Overview

The `Products` component displays a curated selection of featured products in a responsive grid layout. It's designed for the homepage and shows the first 4 products from the product catalog with a call-to-action to view the full catalog.

## Import

```tsx
import { Products } from '../components/Products';
```

## Usage

```tsx
import { Products } from './components/Products';

function HomePage() {
  return (
    <>
      {/* Other sections */}
      <Products />
      {/* More sections */}
    </>
  );
}
```

## Product Grid

The component renders 4 featured products in a responsive grid:

```tsx
const featuredProducts = products.slice(0, 4);

<div className="grid md:grid-cols-2 lg:grid-cols-4 gap-10">
  {featuredProducts.map((p, i) => (
    <ProductCard key={p.id} product={p} delay={i * 100} />
  ))}
</div>
```

<Steps>
  <Step title="Data Selection">
    Extracts first 4 products using `products.slice(0, 4)`
  </Step>
  <Step title="Grid Layout">
    Creates responsive grid: 1 col mobile, 2 cols tablet, 4 cols desktop
  </Step>
  <Step title="Staggered Animation">
    Each card has a 100ms delay multiplied by its index for smooth entrance
  </Step>
</Steps>

## Full Component Code

```tsx
import { products } from '../data/products';
import { ProductCard } from './ProductCard';
import { Link } from 'react-router-dom';

export const Products = () => {
  // Show only first 4 products on home page
  const featuredProducts = products.slice(0, 4);

  return (
    <section 
      id="productos" 
      className="py-32 bg-[#1a1a1a] text-white 
                 rounded-[60px] mx-4 my-10"
    >
      <div className="max-w-7xl mx-auto px-6">
        {/* Header */}
        <div className="flex flex-col md:flex-row justify-between 
                        items-end mb-20 gap-8">
          <div data-aos="fade-right">
            <h2 className="text-5xl md:text-6xl font-serif font-bold mb-4">
              Colección <br /> 
              <span className="text-[#D4AF37]">Exclusiva</span>
            </h2>
            <p className="text-gray-400 max-w-md">
              Piezas limitadas diseñadas para los gustos más exigentes.
            </p>
          </div>
          
          <div className="flex gap-4" data-aos="fade-left">
            <Link 
              to="/catalogo" 
              className="px-8 py-3 rounded-full border border-white/10 
                         hover:bg-white hover:text-black transition-all 
                         text-xs font-bold uppercase tracking-widest"
            >
              Ver Todo el Catálogo
            </Link>
          </div>
        </div>

        {/* Product Grid */}
        <div className="grid md:grid-cols-2 lg:grid-cols-4 gap-10">
          {featuredProducts.map((p, i) => (
            <ProductCard key={p.id} product={p} delay={i * 100} />
          ))}
        </div>
      </div>
    </section>
  );
};
```
