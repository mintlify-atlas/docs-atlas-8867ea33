## Overview

The `About` component (displayed as "Nosotros" in Spanish) presents the brand story with a visually engaging two-column layout featuring an image gallery with stats cards and descriptive content with feature highlights.

## Import

```tsx
import { About } from '../components/About';
```

## Usage

```tsx
import { About } from './components/About';

function HomePage() {
  return (
    <>
      {/* Other sections */}
      <About /> {/* Brand story section */}
      {/* More sections */}
    </>
  );
}
```

## Layout Structure

The component uses a 2-column grid layout on large screens:

<CardGroup cols={2}>

## Styling

### Colors

<ParamField path="Primary Gold" type="color">
  `#D4AF37` - Used for badge, headline accent, icons, and CTA
</ParamField>

<ParamField path="Text Colors" type="color">
  - Primary: Near black (`#1a1a1a`)
  - Secondary: `gray-500`
</ParamField>

<ParamField path="Card Backgrounds" type="color">
  - Gold card: `#D4AF37`
  - Black card: `black`
</ParamField>

### Border Radius

Consistent `rounded-[40px]` (40px radius) on all cards and images for a cohesive look.

### Typography

- **Badge**: `10px` uppercase, extra wide tracking
- **Headline**: `5xl` to `6xl`, serif, bold
- **Body**: `text-lg`, relaxed line height
- **Features**: `text-sm`, bold, tight tracking

## Responsive Behavior

<Steps>
  <Step title="Large Screens (lg+)">
    Two column layout with image gallery on left, content on right
  </Step>
  <Step title="Medium Screens">
    Headline reduces from `6xl` to `5xl`
  </Step>
  <Step title="Small Screens">
    - Single column stacked layout
    - Gallery displays first
    - Features remain in 2-column grid
  </Step>
</Steps>

## Section Padding

```tsx
className="py-32 overflow-hidden"
```

- Vertical padding: `128px` (8rem)
- `overflow-hidden` prevents animation artifacts

## Accessibility

- Semantic heading structure (`<h2>`)
- Descriptive alt text on images
- Clear visual hierarchy
- Sufficient color contrast
- Keyboard accessible button
