## Overview

The `Navbar` component provides a fixed navigation bar with responsive behavior, cart integration, and smooth scroll detection. It features a mobile-friendly hamburger menu and integrates with the CartContext for real-time cart count updates.

## Import

```tsx
import { Navbar } from '../components/Navbar';
```

## Usage

```tsx
import { Navbar } from './components/Navbar';

function App() {
  return (
    <>
      <Navbar />
      {/* Rest of your app */}
    </>
  );
}
```

## Component Structure

### State Management

The component manages three key states:

<ParamField path="isMenuOpen" type="boolean" default="false">
  Controls the visibility of the mobile menu
</ParamField>

<ParamField path="isScrolled" type="boolean" default="false">
  Tracks whether the page has been scrolled past 50px
</ParamField>

<ParamField path="location" type="Location">
  React Router location object for tracking active navigation items
</ParamField>

## Effects

### Menu Open Effect

Prevents body scrolling when mobile menu is open:

```tsx
React.useEffect(() => {
  if (isMenuOpen) {
    document.body.style.overflow = 'hidden';
  } else {
    document.body.style.overflow = 'unset';
  }
  return () => {
    document.body.style.overflow = 'unset';
  };
}, [isMenuOpen]);
```

### Scroll Detection

Changes navbar appearance after 50px scroll:

```tsx
React.useEffect(() => {
  const handleScroll = () => setIsScrolled(window.scrollY > 50);
  window.addEventListener('scroll', handleScroll);
  return () => window.removeEventListener('scroll', handleScroll);
}, []);
```

## Cart Integration

The Navbar uses the CartContext to:

<Steps>
  <Step title="Get Cart Count">
    Displays the number of items in the cart with a badge
  </Step>
  <Step title="Open Cart Drawer">
    Clicking the cart button triggers `setIsCartOpen(true)`
  </Step>
  <Step title="Real-time Updates">
    Cart count badge updates automatically when items are added
  </Step>
</Steps>

```tsx
const { cartCount, setIsCartOpen } = useCart();

<button onClick={() => setIsCartOpen(true)} className="relative">
  <FontAwesomeIcon icon={faBagShopping} />
  {cartCount > 0 && (
    <span className="absolute badge">{cartCount}</span>
  )}
</button>
```
