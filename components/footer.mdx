## Overview

The `Footer` component provides a rich footer section with brand information, navigation links, social media connections, and company details. It features a multi-column layout that's responsive and includes external links to social platforms.

## Import

```tsx
import { Footer } from '../components/Footer';
```

## Usage

```tsx
import { Footer } from './components/Footer';

function App() {
  return (
    <>
      {/* Page content */}
      <Footer />
    </>
  );
}
```

## Responsive Behavior

<Steps>
  <Step title="Large Screens (lg+)">
    - 12-column grid layout
    - 5 columns for brand, 7 for links
    - Horizontal bottom bar
  </Step>
  <Step title="Medium Screens (md)">
    - 3-column link grid
    - Stacked brand section
  </Step>
  <Step title="Small Screens">
    - Single column layout
    - 2-column link grid
    - Stacked bottom bar
  </Step>
</Steps>

## Typography

- **Brand Name**: Font size `3xl`, serif font, bold
- **Section Headings**: `10px` uppercase, wide tracking (`tracking-widest`)
- **Links**: `text-sm` font medium for main links
- **Copyright**: `10px` uppercase, extra wide tracking

## Accessibility Features

- All external links use `target="_blank"` with `rel="noopener noreferrer"`
- Semantic `<footer>` element with `id="contacto"` for anchor navigation
- Clear hover states on all interactive elements
- Descriptive link text

## Dependencies

- `@fortawesome/react-fontawesome` - Icon library
- `@fortawesome/free-solid-svg-icons` - Paw icon
- `@fortawesome/free-brands-svg-icons` - Social media icons
