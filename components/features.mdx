## Overview

The `Features` component displays a grid of four key value propositions using icons and descriptive text. It provides a quick overview of the main benefits customers can expect from Jasper Pet Shop.

## Import

```tsx
import { Features } from '../components/Features';
```

## Usage

```tsx
import { Features } from './components/Features';

function HomePage() {
  return (
    <>
      <Hero />
      <Features /> {/* Feature highlights section */}
      {/* Other sections */}
    </>
  );
}
```

## Component Structure

```tsx
export const Features = () => {
  return (
    <section className="py-20 bg-white">
      <div className="max-w-7xl mx-auto px-6">
        <div className="grid md:grid-cols-4 gap-12">
          {features.map((f, i) => (
            <div key={i} className="group" data-aos="fade-up" 
                 data-aos-delay={i * 100}>
              {/* Icon */}
              <div className="mb-6 w-14 h-14 rounded-2xl bg-[#fcfaf7] 
                              flex items-center justify-center 
                              group-hover:bg-[#D4AF37] 
                              transition-all duration-500 
                              transform group-hover:-rotate-6">
                <FontAwesomeIcon 
                  icon={f.icon} 
                  className="text-[#D4AF37] group-hover:text-white 
                             transition-colors text-2xl" 
                />
              </div>
              
              {/* Title */}
              <h3 className="text-lg font-bold mb-2 tracking-tight">
                {f.title}
              </h3>
              
              {/* Description */}
              <p className="text-sm text-gray-400 leading-relaxed">
                {f.desc}
              </p>
            </div>
          ))}
        </div>
      </div>
    </section>
  );
};
```

## Icon Container

Each feature has an animated icon container:

### Default State

```tsx
className="w-14 h-14 rounded-2xl bg-[#fcfaf7] 
          flex items-center justify-center"
```

- Size: `56x56px` (3.5rem)
- Background: `#fcfaf7` (light cream)
- Border radius: `rounded-2xl` (16px)
- Icon color: `#D4AF37` (gold)

### Hover State

```tsx
group-hover:bg-[#D4AF37]      // Gold background
group-hover:text-white         // White icon
group-hover:-rotate-6          // Slight rotation
transition-all duration-500    // Smooth animation
```

<Info>
  The `-rotate-6` transform creates a playful tilt effect on hover, adding personality to the design.
</Info>

## Animations

Each feature card uses AOS (Animate On Scroll):

```tsx
data-aos="fade-up" 
data-aos-delay={i * 100}
```

### Staggered Animation

The entire card is wrapped in a `group` class, enabling coordinated hover effects:

```tsx

  <div className="group-hover:bg-[#D4AF37] group-hover:-rotate-6">
    <FontAwesomeIcon className="group-hover:text-white" />
  </div>

```

## Customization

### Adding New Features

To add more features, simply extend the array:

```tsx
const features = [
  // Existing features...
  { 
    icon: faLeaf, 
    title: "Eco-Friendly", 
    desc: "Materiales sustentables." 
  },
];
```

### Changing Grid Columns

Adjust the grid for different layouts:

```tsx
// 3 columns
className="grid md:grid-cols-3 gap-12"

// 2 columns
className="grid md:grid-cols-2 gap-12"

// 6 columns
className="grid md:grid-cols-2 lg:grid-cols-6 gap-8"
```

  - `faShieldHalved` - Premium/protection icon

  - `faComment` - Support/communication icon
- AOS library - Scroll animations
