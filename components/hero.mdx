## Overview

The `Hero` component is the landing section of the website featuring a bold headline, call-to-action button, social proof elements, and a featured image. It uses AOS (Animate On Scroll) for entrance animations.

## Import

```tsx
import { Hero } from '../components/Hero';
```

## Usage

```tsx
import { Hero } from './components/Hero';

function HomePage() {
  return (
    <>
      <Hero />
      {/* Other sections */}
    </>
  );
}
```

## Styling

### Typography

- **Headline**: `text-5xl md:text-6xl lg:text-7xl` - Responsive sizing
- **Font**: Serif font family for elegance
- **Tracking**: Tight letter spacing (`tracking-tighter`)
- **Line Height**: `1.1` for compact multi-line headlines

### Colors

- **Background**: `#fcfaf7` - Off-white/cream
- **Primary Gold**: `#D4AF37`
- **Text**: `#1a1a1a` - Near black
- **Secondary Text**: `gray-500`

### Spacing

- **Section Padding**: `pt-32 pb-20` - Accounts for fixed navbar
- **Min Height**: `70vh` - Ensures substantial presence

## Complete Component

```tsx
export const Hero = () => {
  return (
    <section id="inicio" className="relative min-h-[70vh] 
                                    flex items-center pt-32 pb-20 
                                    overflow-hidden bg-[#fcfaf7]">
      {/* Background Accents */}
      <div className="absolute top-0 right-0 w-1/2 h-full 
                      bg-[#D4AF37]/5 -skew-x-12 translate-x-1/4"></div>

      <div className="max-w-6xl mx-auto px-6 grid lg:grid-cols-12 gap-12">
        {/* Content Column */}
        <div className="lg:col-span-7" data-aos="fade-right">
          {/* Badge, Headline, Description, CTA, Social Proof */}
        </div>
        
        {/* Image Column */}
        <div className="lg:col-span-5" data-aos="fade-left" 
             data-aos-delay="200">
          {/* Featured Image */}
        </div>
      </div>
    </section>
  );
};
```
