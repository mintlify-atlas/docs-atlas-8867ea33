## Overview

The `CTA` (Call-to-Action) component provides a prominent conversion section with a bold headline, promotional text, and two action buttons: one for WhatsApp checkout and one to view the catalog.

## Import

```tsx
import { CTA } from '../components/CTA';
```

## Usage

```tsx
import { CTA } from './components/CTA';

function HomePage() {
  return (
    <>
      {/* Other sections */}
      <CTA /> {/* Final conversion section */}
    </>
  );
}
```

<Info>
  This component works best near the end of the page as a final conversion opportunity before the footer.
</Info>

### Button Styles

**Primary Button (WhatsApp)**
- Background: Black
- Hover effect: Scale up to 105%
- Shadow: `shadow-2xl`
- Icon: Right arrow
- Opens in new tab

**Secondary Button (Catalog)**
- Background: `white/20` with backdrop blur (glass effect)
- Hover effect: Increases opacity to `white/30`
- React Router Link (internal navigation)

```tsx
<div className="flex flex-wrap justify-center gap-6">
```

## WhatsApp Link

The WhatsApp button uses the `wa.me` URL format:

```tsx
href="https://wa.me/51903433063"
```

<Info>
  Format: `https://wa.me/[country code][phone number]` (no + or spaces)
</Info>

You can also add a pre-filled message:

```tsx
href="https://wa.me/51903433063?text=Hola!%20Quiero%20información"
```
