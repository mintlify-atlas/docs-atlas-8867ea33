## Overview

Jasper Pet Shop uses two powerful animation libraries to create engaging user experiences:

- **Framer Motion** - Advanced animations for interactive elements
- **AOS (Animate on Scroll)** - Scroll-triggered animations

## AOS Initialization

AOS is initialized in the main App component:

```typescript src/App.tsx
import React from 'react';
import AOS from 'aos';

export default function App() {
    React.useEffect(() => {
        AOS.init({
            duration: 1000,           // Animation duration in ms
            once: true,               // Animation happens only once
            easing: 'ease-out-cubic', // Smooth easing function
            disableMutationObserver: true,
        });
    }, []);

    return (
        // App content
    );
}
```

<Note>
  The `once: true` option prevents animations from repeating on scroll, creating a cleaner experience.
</Note>

### Staggered Animations

Create sequential animations using delays:

```typescript src/components/Features.tsx
const features = [/* feature data */];

return (
    <div className="grid md:grid-cols-4 gap-12">
        {features.map((f, i) => (
            <div 
                key={i} 
                data-aos="fade-up" 
                data-aos-delay={i * 100}  // 0ms, 100ms, 200ms, 300ms
            >
                {/* Feature content */}
            </div>
        ))}
    </div>
);
```

<Tip>
  Multiply the index by 100ms to create smooth staggered effects. Adjust the multiplier to speed up or slow down the stagger.
</Tip>

## Framer Motion Animations

### Cart Drawer Animation

The cart drawer slides in from the right with spring physics:

```typescript src/components/CartDrawer.tsx
import { motion, AnimatePresence } from 'framer-motion';

export const CartDrawer = () => {
    const { isCartOpen } = useCart();

    return (
        <AnimatePresence>
            {isCartOpen && (
                <>
                    {/* Backdrop with fade */}
                    <motion.div
                        initial={{ opacity: 0 }}
                        animate={{ opacity: 1 }}
                        exit={{ opacity: 0 }}
                        className="fixed inset-0 bg-black/40 backdrop-blur-sm"
                    />

                    {/* Drawer with slide + spring */}
                    <motion.div
                        initial={{ x: '100%' }}
                        animate={{ x: 0 }}
                        exit={{ x: '100%' }}
                        transition={{ 
                            type: 'spring', 
                            damping: 25, 
                            stiffness: 200 
                        }}
                        className="fixed top-0 right-0 h-full w-full max-w-md bg-white"
                    >
                        {/* Cart content */}
                    </motion.div>
                </>
            )}
        </AnimatePresence>
    );
};
```

<Note>
  `AnimatePresence` is required for exit animations. It detects when components are removed from the React tree.
</Note>

### Layout Animations

Animate layout changes when filtering products:

```typescript src/pages/Catalog.tsx
import { motion } from 'framer-motion';

export const Catalog = () => {
    const [selectedCategory, setSelectedCategory] = useState('Todos');
    const filteredProducts = /* filter logic */;

    return (
        <motion.div 
            layout  // Automatically animate layout changes
            className="grid md:grid-cols-2 lg:grid-cols-4 gap-10"
        >
            {filteredProducts.map((product, i) => (
                <ProductCard 
                    key={product.id} 
                    product={product} 
                    delay={i * 50} 
                />
            ))}
        </motion.div>
    );
};
```

<Tip>
  The `layout` prop enables automatic FLIP animations when grid items are added, removed, or reordered.
</Tip>

### Hover Animations

Create interactive hover effects with Framer Motion:

```typescript
import { motion } from 'framer-motion';

function Button() {
    return (
        <motion.button
            whileHover={{ scale: 1.05 }}
            whileTap={{ scale: 0.95 }}
            transition={{ type: 'spring', stiffness: 400 }}
        >
            Hover me
        </motion.button>
    );
}
```

### Custom Variants

Define reusable animation variants:

```typescript
const fadeInUp = {
    hidden: { opacity: 0, y: 20 },
    visible: { 
        opacity: 1, 
        y: 0,
        transition: {
            duration: 0.6,
            ease: 'easeOut'
        }
    }
};

<motion.div
    initial="hidden"
    animate="visible"
    variants={fadeInUp}
>
    Content
</motion.div>
```

### Stagger Children

Animate child elements in sequence:

```typescript
const container = {
    hidden: { opacity: 0 },
    visible: {
        opacity: 1,
        transition: {
            staggerChildren: 0.1  // 100ms between each child
        }
    }
};

const item = {
    hidden: { opacity: 0, y: 20 },
    visible: { opacity: 1, y: 0 }
};

<motion.div variants={container} initial="hidden" animate="visible">
    {items.map((item, i) => (
        <motion.div key={i} variants={item}>
            {item.content}
        </motion.div>
    ))}
</motion.div>
```

## Performance Optimization

<Steps>
  <Step title="Use transform properties">
    Animate `transform` (translate, scale, rotate) and `opacity` for 60fps animations. Avoid animating `width`, `height`, or `top/left`.
    
    ```css
    /* Good - uses transform */
    .element {
      transform: translateY(20px);
    }
    
    /* Bad - triggers layout */
    .element {
      top: 20px;
    }
    ```
  </Step>
  
  <Step title="Reduce motion for accessibility">
    Respect user preferences:
    
    ```typescript
    const shouldReduceMotion = window.matchMedia('(prefers-reduced-motion: reduce)').matches;
    
    <motion.div
        animate={shouldReduceMotion ? {} : { scale: 1.1 }}
    >
        Content
    </motion.div>
    ```
  </Step>
  
  <Step title="Use will-change sparingly">
    Hint to the browser for better performance:
    
    ```css
    .animated-element {
      will-change: transform;
    }
    ```
  </Step>
</Steps>
